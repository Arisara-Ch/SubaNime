# SubaNime (In Progress)

**TH:**
เว็บไซต์อนิเมะที่ออกแบบและพัฒนาด้วย React.js  
**ฟีเจอร์ปัจจุบัน:** Navbar, ปุ่มเปลี่ยนโหมดสี, หน้า Home (Responsive)

**Tech Stack:**  
- React.js, Vite  
- CSS, Tailwind / Bootstrap  
- JavaScript, HTML
